# API keys

API tokens are stored in Proton Pass under a flat name (e.g. `hcloud`,
`openrouter`) inside the `api-keys` vault. The password field holds the
token.

## Storing a key

Create or update via the Proton Pass GUI. Use a Login item; put the token in
the password field.

## Retrieving a key

Open the Proton Pass GUI, find the item, and copy the password.
